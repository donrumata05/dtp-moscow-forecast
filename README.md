# Прогнозирование аварийности на дорогах Москвы

Автор: Носов Данила
Дата: 6 июня 2026  
Данные: dtp-stat.ru (открытые данные ГИБДД)

## Задача
Предсказать количество ДТП в ячейках пространственной сетки (550×550 м) на основе погоды, освещения, состояния дороги и координат.

## Этапы работы
1. Очистка и фильтрация данных
2. Разбиение территории Москвы на сетку (7018 ячеек)
3. Feature engineering: one-hot encoding для погоды, освещения, дороги
4. Устранение data leakage - переход от сумм к долям
5. Обучение Random Forest Regressor
6. Визуализация на интерактивной карте

## Результаты
MAE: 3.20 ДТП на ячейку
R²: 0.822 (модель объясняет 82% дисперсии)

## Ключевые выводы
Самые важные факторы: координаты, дождь, ночное освещение
Исправление data leakage позволило модели честно обучаться
